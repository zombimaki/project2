State by State Analysis

The line and bar chart visualizes each state key indicators: home ownership rate and total state gross domestic product. All 15 years of data is reflected in the values. This page allows users to visual data on states and can be used for comparisons.


For example, California’s GDP ($3.1M) is 3924% larger than West Virginia’s GDP ($79K), but West Virginia’s home ownership rate (76%) is 38% greater than California’s (55%).
