API Call Syntax Example
All annual economic data for all states. http://127.0.0.1:5000/api/v1.0/us-state-data http://127.0.0.1:5000/api/v1.0/us-state-data
All annual economic data for a specific state. http://127.0.0.1:5000/api/v1.0/state/[State Name] http://127.0.0.1:5000/api/v1.0/state/California
All state economic data for a specific year. http://127.0.0.1:5000/api/v1.0/year/[Year] http://127.0.0.1:5000/api/v1.0/year/2013
Economic data for a specific year and state. http://127.0.0.1:5000/api/v1.0/state-year/[State Name]/[Year] http://127.0.0.1:5000/api/v1.0/state-year/New York/2015
All State economic data by range of years. http://127.0.0.1:5000/api/v1.0/year-range/[Start Year]/[Last Year] http://127.0.0.1:5000/api/v1.0/year-range/2014/2018
Economic data for a specified state by range of years. http://127.0.0.1:5000/api/v1.0/state-year-range/[State Name]/[Start Year]/[Last Year] http://127.0.0.1:5000/api/v1.0/state-year-range/Colorado/2011/2019
Distinct states in the data set http://127.0.0.1:5000/api/v1.0/distinct_state http://127.0.0.1:5000/api/v1.0/distinct_state
Distinct years in the data set http://127.0.0.1:5000/api/v1.0/distinct_year http://127.0.0.1:5000/api/v1.0/distinct_year
